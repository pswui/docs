# Avatar

Displays a user's profile picture, initials, or custom fallback.

## Playground

<Playground />

## Installation

1. Create a new file `Avatar.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Avatar")} />

## Usage

```tsx
import { Avatar } from "@components/Avatar";
```

```tsx
<Avatar
  src="https://example.com/profile.png"
  alt="Shinwoo Park"
  name="Shinwoo Park"
/>
```

## Props

| Prop         | Type                   | Default      | Description                                        |
| :----------- | :--------------------- | :----------- | :------------------------------------------------- |
| `src`        | `string`               | -            | Image source for the avatar.                       |
| `alt`        | `string`               | -            | Alternative text passed to the image.              |
| `name`       | `string`               | -            | Used to generate initials when no image is shown.  |
| `fallback`   | `React.ReactNode`      | -            | Overrides the default initials fallback.           |
| `imageProps` | `AvatarImageProps`     | -            | Additional props forwarded to the inner `img`.     |
| `size`       | `"sm" \| "md" \| "lg"` | `"md"`       | Controls the avatar dimensions.                    |
| `shape`      | `"circle" \| "square"` | `"circle"`   | Chooses between a circular or rounded square shape. |
