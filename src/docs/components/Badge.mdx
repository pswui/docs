# Badge

Displays a compact status label.

  <TabContent name="code">
    <LoadedCode from={GITHUB_STORY("Badge", "Default")} />
  </TabContent>
</TabProvider>

## Installation

1. Create a new file `Badge.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Badge")} />

## Usage

```tsx
import { Badge } from "@components/Badge";
```

```tsx
<Badge status="success">Paid</Badge>
```

## Props

| Prop      | Type                                              | Default     | Description                               |
| :-------- | :------------------------------------------------ | :---------- | :---------------------------------------- |
| `size`    | `"sm" \| "md"`                                    | `"md"`      | Controls the badge padding and text size. |
| `status`  | `"default" \| "success" \| "warning" \| "danger"` | `"default"` | Applies the matching status colors.       |
| `asChild` | `boolean`                                         | `false`     | Renders the badge through a child slot.   |
