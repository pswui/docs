## Usage

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationEllipsis,
  PaginationItem,
  PaginationLink,
  PaginationNext,
  PaginationPrevious,
} from "@components/Pagination";
```

```tsx
<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="#previous" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#page-1" active>
        1
      </PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationEllipsis />
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#next" />
    </PaginationItem>
  </PaginationContent>
</Pagination>
```

## Props

### Pagination

Standard `nav` props plus `asChild`. `aria-label` defaults to `"Pagination"`.

### PaginationContent

Standard `ul` props plus `asChild`.

### PaginationItem

Standard `li` props plus `asChild`.

### PaginationLink

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `active` | `boolean` | `false` | Marks the current page and sets `aria-current="page"` by default. |
| `disabled` | `boolean` | `false` | Disables interaction, removes `href`, and prevents clicks. |
| `asChild` | `boolean` | `false` | Render the link through `Slot`. |

### PaginationPrevious / PaginationNext

Both components inherit `PaginationLink` props and accept optional `children` to replace the default labels.

### PaginationEllipsis

Standard `span` props plus `asChild`. It renders `"..."` by default.
