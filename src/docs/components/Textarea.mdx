## Installation

1. Create a new file `Textarea.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Textarea")} />

## Usage

```tsx
import { Textarea, TextareaFrame } from "@components/Textarea";
```

```tsx
<Textarea
  className="min-h-28"
  placeholder="Type your message..."
/>
```

## Props

### Textarea

| Prop       | Type      | Default | Description                                          |
| :--------- | :-------- | :------ | :--------------------------------------------------- |
| `invalid`  | `string`  | -       | Sets the browser validity message and `aria-invalid`. |
| `unstyled` | `boolean` | `false` | Removes the default field styling.                   |
| `full`     | `boolean` | `false` | Makes the textarea take the full available width.    |

### TextareaFrame

Accepts the same `unstyled` and `full` variants as `Textarea`, plus standard
label props and `asChild`.
