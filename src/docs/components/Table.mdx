## Installation

1. Create a new file `Table.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Table")} />

## Usage

```tsx
import {
  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableFooter,
  TableHead,
  TableHeader,
  TableRow,
} from "@components/Table";
```

```tsx
<Table>
  <TableCaption>Caption</TableCaption>
  <TableHeader>
    <TableRow>
      <TableHead>Header</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Cell</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

## Props

### Table

Accepts standard table props.

### TableHeader

Accepts standard `thead` props.

### TableBody

Accepts standard `tbody` props.

### TableFooter

Accepts standard `tfoot` props.

### TableRow

Accepts standard `tr` props. Set `data-state="selected"` to apply the selected
row styling.

### TableHead

Accepts standard `th` props.

### TableCell

Accepts standard `td` props.

### TableCaption

Accepts standard `caption` props.
