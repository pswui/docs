## Installation

1. Create a new file `Slider.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Slider")} />

## Usage

```tsx
import { Slider } from "@components/Slider";
```

```tsx
<Slider
  min={0}
  max={100}
  step={5}
  defaultValue={50}
/>
```

## Props

| Prop   | Type                    | Default | Description                                  |
| :----- | :---------------------- | :------ | :------------------------------------------- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"`  | Controls the rendered height of the slider. |

The component also accepts standard `input[type="range"]` props such as
`min`, `max`, `step`, `value`, `defaultValue`, and `disabled`.
