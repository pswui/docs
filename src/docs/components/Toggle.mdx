# Toggle

A two-state button that toggles between pressed and not pressed.

## Playground

<Playground />

## Installation

1. Create a new file `Toggle.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Toggle")} />

## Usage

```tsx
import { Toggle } from "@components/Toggle";
```

```tsx
<Toggle defaultPressed>Bold</Toggle>
```

## Props

| Prop              | Type                          | Default | Description                                 |
| :---------------- | :---------------------------- | :------ | :------------------------------------------ |
| `size`            | `"sm" \| "md" \| "lg"`       | `"md"`  | Controls the toggle padding and text size.  |
| `pressed`         | `boolean`                    | -       | Controls the pressed state externally.      |
| `defaultPressed`  | `boolean`                    | `false` | Sets the initial uncontrolled pressed state. |
| `onPressedChange` | `(pressed: boolean) => void` | -       | Called whenever the pressed state changes.  |
