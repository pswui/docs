Displays the current location within a hierarchy of links.

## Installation

1. Create a new file `Breadcrumb.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Breadcrumb")} />

## Usage

```tsx
import {
  Breadcrumb,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbList,
  BreadcrumbPage,
  BreadcrumbSeparator,
} from "@components/Breadcrumb";
```

```tsx
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="#">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## Props

### Breadcrumb

Accepts standard `nav` props. The component sets `aria-label="Breadcrumb"` by
default.

### BreadcrumbList

Accepts standard ordered list props.

### BreadcrumbItem

Accepts standard list item props.

### BreadcrumbLink

Accepts standard anchor props plus `asChild`.

### BreadcrumbPage

Accepts standard span props plus `asChild`. It sets `aria-current="page"` by
default.

### BreadcrumbSeparator

Accepts standard list item props. When no children are provided it renders `/`.
