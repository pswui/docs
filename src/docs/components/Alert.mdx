# Alert

Displays a callout for important information.

## Playground

<Playground />

## Installation

1. Create a new file `Alert.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Alert")} />

## Usage

```tsx
import { Alert, AlertDescription, AlertTitle } from "@components/Alert";
```

```tsx
<Alert status="warning">
  <AlertTitle>Review your changes</AlertTitle>
  <AlertDescription>
    This action updates data for everyone in the workspace.
  </AlertDescription>
</Alert>
```

## Props

### Alert

| Prop      | Type                                              | Default     | Description                             |
| :-------- | :------------------------------------------------ | :---------- | :-------------------------------------- |
| `status`  | `"default" \| "success" \| "warning" \| "danger"` | `"default"` | Applies the corresponding alert styles. |
| `asChild` | `boolean`                                         | `false`     | Renders the alert through a child slot. |

### AlertTitle

| Prop      | Type      | Default | Description                             |
| :-------- | :-------- | :------ | :-------------------------------------- |
| `asChild` | `boolean` | `false` | Renders the title through a child slot. |

### AlertDescription

| Prop      | Type      | Default | Description                                   |
| :-------- | :-------- | :------ | :-------------------------------------------- |
| `asChild` | `boolean` | `false` | Renders the description through a child slot. |
