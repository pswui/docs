# ScrollArea

A keyboard-focusable scroll container with hidden native scrollbars and custom thumb-only overlay chrome.

## Playground

<Playground />

## Installation

1. Create a new file `ScrollArea.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("ScrollArea")} />

## Usage

```tsx
import { ScrollArea } from "@components/ScrollArea";
```

```tsx
<ScrollArea
  role="region"
  aria-label="Activity feed"
  className="h-56 rounded-lg border border-neutral-200 p-3 pr-5"
>
  <div className="flex flex-col gap-3 pr-2">
    <p>Scrollable content...</p>
  </div>
</ScrollArea>
```

The overlay thumb fades after a short idle delay, reappears on focus, scroll, hover, or pointer movement, and can be dragged directly. Because the chrome sits on top of the content edge, add inner padding on the scrolling edge when needed.

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `orientation` | `"vertical" \| "horizontal" \| "both"` | `"vertical"` | Controls which axis can scroll and which overlay thumb is rendered. |

`ScrollArea` also defaults `tabIndex` to `0`, so keyboard users can focus it and scroll with the keyboard.
