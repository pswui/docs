# ScrollArea

A scrollable container with subtle, styled scrollbars.

## Playground

<Playground />

## Installation

1. Create a new file `ScrollArea.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("ScrollArea")} />

## Usage

```tsx
import { ScrollArea } from "@components/ScrollArea";
```

```tsx
<ScrollArea className="h-40 w-80">
  <p>Scrollable content...</p>
</ScrollArea>
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `orientation` | `"vertical" \| "horizontal" \| "both"` | `"vertical"` | Controls which axis can scroll. |

`ScrollArea` also defaults `tabIndex` to `0`, so keyboard users can focus it and scroll with the keyboard.
