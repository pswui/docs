Shows the completion state of a task with a native progress element.

## Installation

1. Create a new file `Progress.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Progress")} />

## Usage

```tsx
import { Progress } from "@components/Progress";
```

```tsx
<Progress value={64} max={100} aria-label="Upload progress" />
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Controls the height of the progress bar. |
| `max` | `number` | `100` | Upper bound for the progress value. Invalid or non-positive values fall back to `100`. |
| `value` | `number \| null` | - | Current progress value. Leave it unset to render an indeterminate state. |
