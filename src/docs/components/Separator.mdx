# Separator

Displays a visual divider between related content.

<TabProvider defaultName="preview">
  <TabList>
    <TabTrigger name="preview">Preview</TabTrigger>
    <TabTrigger name="code">Code</TabTrigger>
  </TabList>
  <TabContent name="preview">
    <Story layout="centered">
      <Examples.Default />
    </Story>
  </TabContent>
  <TabContent name="code">
    <LoadedCode from={GITHUB_STORY("Separator", "Default")} />
  </TabContent>
</TabProvider>

## Installation

1. Create a new file `Separator.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Separator")} />

## Usage

```tsx
import { Separator } from "@components/Separator";
```

```tsx
<Separator orientation="horizontal" />
```

## Props

| Prop          | Type                         | Default        | Description                                          |
| :------------ | :--------------------------- | :------------- | :--------------------------------------------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Sets whether the separator is horizontal or vertical. |
| `decorative`  | `boolean`                    | `false`        | Hides the separator from assistive technology.       |
