# Form

Composable form primitives for labels, helper text, and validation messages.

<Playground />

## Installation

1. Create a new file `Form.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Form")} />

## Usage

```tsx
import { FormError, FormHelper, FormItem, FormLabel } from "@components/Form";
import { Input } from "@components/Input";
```

```tsx
<FormItem invalid="Email is required.">
  <FormLabel>Email</FormLabel>
  <Input type="email" placeholder="name@example.com" />
  <FormHelper hiddenOnInvalid>Use the address you check every day.</FormHelper>
  <FormError />
</FormItem>
```

## Props

### FormItem

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `invalid` | `string \| null \| undefined` | - | Validation message shared with `FormError` and applied to the nested input element. |
| `asChild` | `boolean` | `false` | Render the wrapper through `Slot` instead of the default `label` element. |

### FormLabel

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `asChild` | `boolean` | `false` | Render the label text through `Slot`. |

### FormHelper

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `hiddenOnInvalid` | `boolean` | `false` | Hide helper text while the parent `FormItem` is invalid. |
| `asChild` | `boolean` | `false` | Render the helper through `Slot`. |

### FormError

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `asChild` | `boolean` | `false` | Render the error container through `Slot`. |

`FormError` reads the `invalid` value from the closest `FormItem` and renders nothing when no message is present.
