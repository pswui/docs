# Card

Displays content and actions inside a bordered container.

## Playground

<Playground />

## Installation

1. Create a new file `Card.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Card")} />

## Usage

```tsx
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from "@components/Card";
```

```tsx
<Card>
  <CardHeader>
    <CardTitle>Team plan</CardTitle>
    <CardDescription>For growing product teams</CardDescription>
  </CardHeader>
  <CardContent>Shared projects, analytics, and billing controls.</CardContent>
  <CardFooter>Footer actions</CardFooter>
</Card>
```

## Props

### Card

Accepts standard `article` props plus `asChild`.

### CardHeader

Accepts standard `header` props plus `asChild`.

### CardTitle

Accepts standard heading props plus `asChild`.

### CardDescription

Accepts standard paragraph props plus `asChild`.

### CardContent

Accepts standard div props plus `asChild`.

### CardFooter

Accepts standard `footer` props plus `asChild`.
