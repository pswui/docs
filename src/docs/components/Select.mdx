# Select

An accessible custom select with keyboard navigation and controlled or uncontrolled state.

## Playground

<Playground />

## Installation

1. Create a new file `Select.tsx` in your component folder.
2. Copy and paste the following code into the file.

<LoadedCode from={GITHUB_COMP("Select")} />

## Usage

```tsx
import { Select, type SelectOption } from "@components/Select";

const options: SelectOption[] = [
  { label: "React", value: "react" },
  { label: "Vue", value: "vue" },
];
```

```tsx
<Select options={options} placeholder="Choose a framework" />
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `options` | `SelectOption[]` | - | List of options rendered in the popup listbox. |
| `value` | `string` | - | Controlled selected value. |
| `defaultValue` | `string` | - | Initial value for uncontrolled usage. |
| `onValueChange` | `(value: string) => void` | - | Called after a new option is committed. |
| `placeholder` | `React.ReactNode` | `"Select an option"` | Content shown when no option is selected. |
| `name` | `string` | - | Writes the selected value into a hidden input for form submission. |
| `full` | `boolean` | `false` | Makes the trigger stretch to the width of its container. |

### SelectOption

```ts
interface SelectOption {
  disabled?: boolean;
  label: React.ReactNode;
  value: string;
}
```
